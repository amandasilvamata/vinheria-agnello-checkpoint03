# Vinheria Agnello - Checkpoint 03

## Descrição
Sistema de gerenciamento de vinhos desenvolvido em JavaScript para o Checkpoint 03.

O sistema realiza cadastro de vinhos, controle de estoque e classificação por antiguidade utilizando objetos, arrays e funções de alta ordem.

## Funcionalidades
- Cadastro de vinhos
- Listagem completa de vinhos
- Verificação de estoque baixo
- Cálculo de estoque total
- Exibição dos nomes em caixa alta
- Classificação de vinhos por idade

## Tecnologias Utilizadas
- HTML5
- CSS3
- JavaScript

## Conceitos Aplicados
- Arrays
- Objetos
- Funções
- forEach
- map
- filter
- reduce
