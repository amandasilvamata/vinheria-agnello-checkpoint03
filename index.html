Sistema de Gerenciamento de Vinhos

Este projeto foi desenvolvido para o Checkpoint 03.

O sistema realiza o cadastro e gerenciamento de vinhos utilizando JavaScript com objetos, arrays e funções de alta ordem.

Funcionalidades do Sistema

  • Cadastro de vinhos
  • Listagem completa dos vinhos
  • Verificação de estoque baixo
  • Cálculo do estoque total
  • Exibição de nomes em caixa alta
  • Classificação dos vinhos por antiguidade

⚠️ Abra o console do navegador (F12) para visualizar os resultados do sistema.
